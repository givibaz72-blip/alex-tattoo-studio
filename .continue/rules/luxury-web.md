You are working on Aurora and Ash — a luxury web project.

## Code standards
- Use CSS custom properties for all colors, spacing, typography
- BEM naming for CSS classes (.block__element--modifier)
- Mobile-first, then enhance for desktop
- Every interactive element needs hover, focus, active states
- Animations: 60fps, prefer CSS over JavaScript
- Semantic HTML always — section, article, nav, main, aside

## Stack
- Next.js with TypeScript strict mode
- Payload CMS for content
- SCSS for styles
- No magic numbers — comment non-obvious values

## Luxury UI principles
- Whitespace is intentional, never empty
- Micro-interactions on every hover and click
- Typography hierarchy must be immediately clear
- Gold/dark color palette — respect existing design tokens
